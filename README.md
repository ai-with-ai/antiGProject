# Base Project

Central repository of reusable skills, workflows, rules, and agents for the Antigravity workspace. This project serves as the **standard foundation** for all new projects.

## What's Included

| Type | Count | Description |
|------|-------|-------------|
| Skills | 11 | Reusable Claude Code capabilities |
| Workflows | 3 | CI/CD and local infrastructure templates |
| Rules | 1 | Architectural constraint sets |
| Agents | 4 | Specialized agent definitions |

## Tech Stack

- **Backend:** Java 21 · Spring Boot 3 · Maven · Hibernate
- **Frontend:** Angular 21 · TypeScript strict mode
- **Database:** MySQL 8.0+
- **Infrastructure:** Docker · Docker Compose · GitHub Actions

## Skills

- `angular-best-practices` — Angular v17+ performance patterns (Signals, OnPush, @defer)
- `database-design` — Schema design, indexing, ORM selection, migrations
- `find-skills` — Discover and install skills from the open ecosystem
- `github-automation` — Automate GitHub issues, PRs, and repositories
- `hibernate-mysql-pro` — Hibernate + MySQL 8 optimization
- `java-spring-ddd` — Java 21 / Spring Boot 3 with 4-layer DDD
- `programmatic-seo` — SEO at scale using templates and data
- `project-intro` — Auto-generate a live project overview
- `project-planner` — Orchestrate project bootstrapping
- `prompt-generator` — Expert prompt engineering
- `skill-creator` — Create new workspace skills

## Workflows

- `maven-github-ci` — Java 21 / Maven build and test via GitHub Actions
- `angular-github-ci` — Angular 21 lint and build via GitHub Actions
- `docker-compose-infra` — Local development with Docker Compose

## Rules

- `ddd-layer-rules` — 4-layer DDD architectural constraints

## Agents

- `backend-ddd-lead` — Java / Spring Boot / DDD specialist
- `frontend-spa-lead` — Angular SPA specialist
- `mysql-dba-lead` — MySQL 8 database specialist
- `devops-infra-lead` — Docker / GitHub Actions specialist
